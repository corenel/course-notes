\begin{itemize}

  \item 计算(4-5题): 会有类似于表格计算那种
  \item 没有证明题

\section{考试内容}
\begin{itemize}
  \item 第一章: 简单概念
  \item 第二章, 第八章不考
  \item 第三章: 第二节与第三节,高斯消去法,LU分解,以及一些概念性的东西(判断题, 如条件数), 大型矩阵分解不考
  \item 第四章: 牛顿与拉格朗日插值法, 牛顿插值的特点, 曲线拟合(正交多项式的拟合)
  \item 第五章: 梯形公式, 复化, Simpson公式, 数值积分(求积分形式), 数值微分(看一下就好), 外推原理与计算
  \item 第六章: 第2,3,4节(RK法, 线性多步法, 什么是打靶法, 差分法)
  \item 第七章: 一元方程基本迭代, 牛顿迭代
  \item 第九章: QR分解, J和H, 幂法(不考计算)
\end{itemize}
